CSS

Early Web Development (1990s):

In the early days of the World Wide Web, websites were simple and primarily consisted of text. Styling was done directly within HTML using attributes like and . As web development advanced, there was a growing need for a separate language to handle the presentation aspects, leading to the development of CSS.

CSS Level 1 (1996):

The first official CSS specification, CSS Level 1, was introduced in December 1996 by the World Wide Web Consortium (W3C). CSS Level 1 included basic styling properties for fonts, colors, and text alignment.

CSS Level 2 (1998):

CSS Level 2, published in May 1998, brought significant improvements and expanded capabilities. It introduced features like positioning, z-index, and the ability to apply styles to different media types (screen, print, etc.). The separation of content and presentation became more pronounced with CSS Level 2..

History to development

In 1980, physicist Tim Berners-Lee, a contractor at CERN, proposed and prototyped ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee wrote a memo proposing an Internet-based hypertext system.[3] Berners-Lee specified HTML and wrote the browser and server software in late 1990. That year, Berners-Lee and CERN data systems engineer Robert Cailliau collaborated on a joint request for funding, but the project was not formally adopted by CERN. In his personal notes of 1990, Berners-Lee listed "some of the many areas in which hypertext is used"; an encyclopedia is the first entry.[4] The first publicly available description of HTML was a document called "HTML Tags",[5] first mentioned on the Internet by Tim Berners-Lee in late 1991.[6][7] It describes 18 elements comprising the initial, relatively simple design of HTML. Except for the hyperlink tag, these were strongly influenced by SGMLguid, an in-house Standard Generalized Markup Language (SGML)-based documentation format at CERN. Eleven of these elements still exist in HTML 4.

Facts

HTML

HTML (Hypertext Markup Language) is used to create the content of a page, such as text. HTML provides the basic structure of a website.

CSS

CSS (Cascade Styling Sheets) is used to control the style of a website, including layout, visual effects, and background color. CSS is used to control presentation, formatting, and layout.
